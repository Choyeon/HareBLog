# HareBlog

#### 介绍
一个多人博客系统

#### 运行要求

#### 快速运行
内置了Pipfile文件推荐使用pipenv
pipenv安装
```shell script
# Windows系统
pip3 install pipenv
# Mac OS 
brew install pipenv
```
运行
```shell
cd HareBlong
pipenv install 
pipenv shell 
python3 ./manage.py makemigrations
python3 ./manage.py migrate
python3 ./manage.py runserver
```
